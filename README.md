# WordPress Snippets

Este repositório armazena snippets de código utilizados com o plugin WPCode no WordPress.

## Objetivo
O objetivo principal é manter o versionamento e o histórico de alterações dos códigos inseridos no site. Como o plugin WPCode armazena os snippets diretamente no banco de dados, este repositório serve como um backup estruturado e uma forma de rastrear modificações futuras.

## Como utilizar
Os arquivos estão organizados por funcionalidade. Para aplicar, basta copiar o conteúdo do arquivo desejado e colar em um novo snippet dentro do plugin WPCode no painel administrativo do WordPress.
